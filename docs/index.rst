agepy
=====

|versionshield| |licenseshield| |testshield|

*agepy* is the python package of the Arbeitsgruppe Ehresmann, which ...

* is intended as a platform for sharing and discussing code used in our 
  group.
* provides a starting point for new group members.
* standardizes plotting styles, etc.
* provides solutions to common data evaluation problems.
* distributes the *evaluix* GUI.

Here you can find documentation on the :doc:`installation` of the
*agepy* package.
Each implemented function is listed and documented in the 
:doc:`reference` section and :doc:`tutorials` are available on some 
topics.
The source code can be viewed on `GitHub`_ and there is a tutorial on 
how to :doc:`contribute` to the package.
